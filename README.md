# Автоскринер очков семьи (Advance RP)

Система автоматического учета очков участников семьи в GTA SA-MP (сервер Advance RP), ведения базы долгов и аудита выплат через Telegram-бота.

---

## 🚀 Как настроить и запустить (для Создателя семьи)

### Шаг 1: Создание Telegram-бота
1. Откройте Telegram и найдите официального бота **[@BotFather](https://t.me/BotFather)**.
2. Введите команду `/newbot` и следуйте инструкциям для создания бота.
3. Скопируйте полученный **Token** (выглядит как `123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ`).
4. Узнайте свой личный Telegram ID (например, через бот [@userinfobot](https://t.me/userinfobot)) для управления ботом в личных сообщениях.

### Шаг 2: Настройка конфигурации
1. В папке с проектом откройте файл **`.env`** любым текстовым редактором.
2. Заполните поля:
   - `TELEGRAM_TOKEN` — вставьте токен вашего бота.
   - `ADMIN_IDS` — вставьте ваш личный Telegram ID (чтобы управлять ботом в личке).
   - `SECRET_TOKEN` — придумайте секретный пароль для авторизации входящих запросов из игры.
   - `POINT_RATE` — цена 1 очка в виртах (по умолчанию `10000`$).
   - (Поля `FAMILY_CHAT_ID` и `FAMILY_NAME` можно оставить пустыми — они настроятся прямо из Telegram).

### Шаг 3: Запуск бота
1. Запустите файл **`run_bot.bat`** двойным кликом.
2. Скрипт сам автоматически проверит и установит нужные библиотеки Python (`Flask`, `pyTelegramBotAPI`, `python-dotenv`), после чего запустит сервер и бота.
3. Сверните окно консоли. Бот готов к приему данных!

### Шаг 4: Привязка чата и имя семьи в Telegram
1. Добавьте созданного бота в ваш групповой чат семьи в Telegram.
2. Назначьте бота администратором группы (чтобы он мог читать сообщения с командами).
3. В чате группы отправьте команду:
   * **`/setchat`** — привяжет этот чат для отчетов. Бот будет отправлять сюда ежедневные скрининги очков.
   * **`/setfamily Casey`** — установит название вашей семьи (например, *Casey*). В отчетах будет красиво писаться «Семья Casey».
4. **Права администраторов**: Бот **автоматически** считает администраторами всех, кто является администратором вашей группы в Telegram! Вам не нужно вручную прописывать Telegram ID ваших заместителей в конфигурацию — просто выдайте им админку в группе Telegram, и они смогут пользоваться ботом.

## 🎮 Как установить и настроить в игре (для Замов и Создателя)

### Шаг 1: Установка скрипта
1. У вас должен быть установлен **MoonLoader v0.26+** и библиотека **SAMP.Lua** (если вы используете *Universal Helper Platform*, они у вас уже есть).
2. Скопируйте файл `family_screener.lua` в вашу папку `GTA San Andreas/moonloader/`.

### Шаг 2: Настройка подключения скрипта к серверу
1. Запустите игру один раз, чтобы скрипт создал файл настроек в папке `GTA San Andreas/moonloader/config/family_screener_config.json`.
2. Откройте созданный файл настроек и укажите:
   - `server_url` — адрес вашего сервера. Если бот запущен на том же ПК, оставьте `http://localhost:5000`. Если бот на хостинге, укажите `http://IP_АДРЕС_СЕРВЕРА:5000`.
   - `secret_token` — вставьте тот же секретный пароль, который вы указали в файле `.env` бота.
   - `auto_scan_after_20h` — `true` (автоматически сканировать вечером после 20:00 при входе в игру) или `false` (только вручную).

## 📋 Инструкция по использованию

### В игре:
* **`/fscan`** — запустить ручной скрининг очков. Скрипт автоматически откроет меню `/family` -> «Участники семьи», считает все данные очках, закроет меню (всё это происходит за 0.2 секунды и невидимо для вас) и отправит данные боту в Telegram.
* **Авто-вечерний скрининг**: Если опция `auto_scan_after_20h` включена, при входе в игру после 20:00 скрипт автоматически выполнит скрининг 1 раз за день.
* **💵 Авто-списание долгов (Smart Pay)**: Когда вы переводите игроку деньги прямо в игре через команды `/pay [ID] [Сумма]` или `/transfer [ID] [Сумма]`, скрипт перехватывает это действие, определяет ник игрока и шлет сигнал боту. Бот автоматически спишет долг игрока в базе данных по текущему курсу и пришлет уведомление в Telegram!

### В Telegram (команды для администраторов группы / владельца бота):
* `/stats` (или `/members`) — Показать актуальный список игроков, у которых есть неоплаченные очки, с суммой долга в виртах.
* `/pay [Ник_Игрока] [количество]` — Зафиксировать выплату игроку:
  - `/pay Maximo_Himars` — Оплатить весь долг игрока.
  - `/pay Maximo_Himars 10p` — Списать ровно 10 очков.
  - `/pay Maximo_Himars 100k` — Оплатить 100,000$ (автоматически переведет в очки по курсу).
* `/payall` — Выплатить долги всем участникам сразу (обнулить долги в базе).
* `/audit [Ник_Игрока]` — История активности игрока: ранг, сколько очков заработано, оплачено очков и история последних 5 выплат.
* `/rename [Старый_Ник] [Новый_Ник]` — Сменить ник игрока в базе данных с переносом всей его истории очков и выплат (критично при смене ников в игре!).
* `/setrate [курс]` — Сменить цену 1 очка (например, `/setrate 15000`).
* `/setchat` — Привязать текущую группу для отправки отчетов по очкам.
* `/setfamily [Имя]` — Установить название семьи.
* `/status` — Общая статистика по семье: сколько всего очков заработано за всё время, сколько выплачено, общий текущий долг семьи и дата последнего скрининга.
